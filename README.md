# contacts-brevo

Este formulário coleta contatos para sorteio em eventos do INAFF - 2025

## Framework

- Nextjs

## Linguagem

- Typescript

## Versão do node

- 20.17.0

## Versão do npm

- 10.8.2

## Como rodar esse projeto

1. Clone o projeto
   `git clone https://github.com/inaff-io/sorteio-evento-2025.git`

2. Instale as dependências
   ` npm install`

3. Rode o projeto
   ` npm run dev`
